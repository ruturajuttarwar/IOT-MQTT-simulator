# IoT MQTT Simulation Dashboard - Frontend

React-based frontend for the IoT MQTT simulation dashboard.

## Features

- Real-time network visualization
- Node management (add/delete nodes dynamically)
- MQTT message logging with QoS support
- Energy monitoring
- Mobility tracking
- MAC layer statistics
- Performance metrics

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist` directory.

## Technology Stack

- React 18
- Vite
- Tailwind CSS
- Socket.IO Client
- Lucide React (icons)
