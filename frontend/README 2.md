# IoT/MQTT Dashboard - React Frontend

Modern React frontend with Tailwind CSS for the IoT/MQTT simulation.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- Tailwind CSS
- Vite
- Socket.IO Client
- Lucide React (icons)

## Development

The frontend runs on `http://localhost:3000` and proxies API requests to the Flask backend on `http://localhost:8000`.
